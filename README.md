# 这个圈子 - H5 Web 增强版

单文件 `index.html` 的前端演示页面（不依赖后端）。

## 功能亮点
- 明亮毛玻璃风格 UI
- 本地持久化（`localStorage`）：加入圈子、聊天、发帖、点赞/评论、活动 RSVP、通知已读
- 桌面宽屏适配（更适合电脑截面浏览）
- GitHub Pages 直接部署即可访问

## 部署到 GitHub Pages
将仓库根目录的 `index.html` 上传后：
1. GitHub 仓库页面：`Settings` -> `Pages`
2. Source 选择 `Deploy from a branch`
3. Branch 选择 `main`（或 `master`）并选择 `/ (root)`
4. 保存后等待生成链接
